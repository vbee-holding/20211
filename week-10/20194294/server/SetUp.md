# Cách setup môi trường để chạy chương trình
## Bước 1. Pull code từ repo về và chạy lệnh npm install để cài đặt các thư viện còn thiếu
![image](https://user-images.githubusercontent.com/58870427/141327673-adf64d34-0bb9-4b88-a876-b82ffe48ade8.png)
## Bước 2. Sử dụng lệnh: nodemon index.js ( nếu như máy đã có thư viện nodemon) hoặc lệnh node index.js để chạy chương trình
![image](https://user-images.githubusercontent.com/58870427/141328039-adc0f7f1-c62b-432c-938a-9f0988bcca9d.png)
Server đã được chạy ở http://localhost:4000
